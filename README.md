# 🔥 Guy — AI Personal Assistant Showcase

> *"The flames of youth burn eternal. Let's get to work."*

A showcase page for **Guy**, a personal AI agent built on [OpenClaw](https://openclaw.ai) that lives on your server, knows your life, and handles your world — 24/7, proactively, without being asked.

## Live Demo

🌐 **[guy-showcase.vercel.app](https://guy-showcase.vercel.app)**

## What's Guy?

Guy is not a chatbot. Guy is an autonomous agent that:

- 📰 Sends a **morning news brief** every day at 5:30 AM (tech headlines, business ideas, upcoming sports/holidays)
- 🏛️ Delivers **daily Stoic wisdom** from Marcus Aurelius, Epictetus, and Seneca
- 📖 Teaches an **advanced vocabulary word** every afternoon
- 🎓 Runs an **adaptive OMSCS quiz** with calibration, spaced repetition, and difficulty scaling
- 🔧 Performs **weekly maintenance** on the server and reports back
- 💬 Communicates entirely through **Telegram** — your phone is the terminal

## Tech Stack

- **Runtime:** [OpenClaw](https://openclaw.ai) — personal AI agent framework
- **Model:** Claude (Anthropic)
- **Hosting:** Self-hosted Linux VPS
- **Messaging:** Telegram Bot API
- **Frontend:** Vanilla HTML/CSS/JS — no frameworks
- **Deployment:** Vercel (auto-deploy on push)
